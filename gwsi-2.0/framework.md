# GWSI 2.0 框架详解

## 1. 九维度评分体系详解

### 维度1: 市场规划力 (25%)
**定义**: 市场未来10年的发展潜力和投资吸引力

**评分维度**:
- 装机容量CAGR (2026-2035)
- 政府投资承诺度
- 产业政策支持力度
- 市场竞争强度
- 技术创新活跃度

**评分标准**:
```
80-100: 年均增长率>15%，政府大力支持
60-79:  年均增长率 10-15%，政策稳定
40-59:  年均增长率 5-10%，政策波动
20-39:  年均增长率 2-5%，政策不确定
0-19:   年均增长率<2%，市场衰退
```

---

### 维度2: 进入壁垒 (15%)
**定义**: 市场进入的难度（评分越低=壁垒越高）

**评分维度**:
- 贸易壁垒（关税、配额、本地化要求）
- 技术标准要求
- 现有竞争格局
- 必要的许可证/认证数量
- 市场准入时间

**评分标准**:
```
80-100: 完全开放市场，无特殊壁垒
60-79:  部分壁垒，容易克服
40-59:  中等壁垒，需要合作伙伴
20-39:  高壁垒，需要合资或代理
0-19:   严格壁垒，基本无法独立进入
```

---

### 维度3: 基础设施 (10%)
**定义**: 风电产业所需的物理基础设施完善度

**评分维度**:
- 风资源质量（平均风速、年均发电小时数）
- 电网建设水平（消纳能力、稳定性）
- 物流运输能力
- 工业园区/制造基地
- 通信网络覆盖

**评分标准**:
```
80-100: 风资源优，电网发达，物流完善
60-79:  风资源中等，电网较好
40-59:  风资源一般，电网可扩展
20-39:  风资源有限，电网待完善
0-19:   风资源差，基础设施不足
```

---

## 2. 修正系数体系

### 2.1 基础修正系数 (数据可靠性)

```
真实指标 (IPyNAME, ALMM, KTVA等):     1.10-1.20
├─ 完整、准确、实时的官方数据
├─ 第三方专业机构认证
└─ 系数取决于数据完整性

行业标准指标:                          1.00-1.10
├─ IEC国际标准
├─ 各国政府官方统计
└─ 第三方审计报告

分析估算:                              0.70-0.85
├─ 基于样本推导
├─ 历史数据趋势
└─ 专家判断
```

### 2.2 增分组项

| 增分项 | 加分范围 | 条件 |
|------|--------|------|
| **风资源优势** | +0.00-0.20 | 平均风速≥7.5m/s，年均发电小时数≥2500 |
| **电网友好性** | +0.00-0.20 | 消纳率>95%，电网稳定性指数>0.95 |
| **产业成熟度** | +0.00-0.20 | 完整的产业链配套，本地人才储备充足 |

### 2.3 综合回报率

$$修正系数 = 基础修正系数 × \prod(1 + 增分项)$$

**修正系数范围: 0.40 - 1.20**

## 3. GWSI 评分计算公式

### 3.1 基础GWSI评分

$$GWSI_{Base} = \sum_{i=1}^{9} (Score_i × Weight_i)$$

其中:
- $Score_i$ = 第i维度的原始得分 (0-100)
- $Weight_i$ = 第i维度的权重

### 3.2 修正后GWSI评分

$$GWSI = GWSI_{Base} × CorrectionFactor$$

### 3.3 评分解释

```
GWSI Score 解释:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
80-100  ⭐⭐⭐  优先进入级
        特点: 市场机会大，风险可控
        建议: 加大投资，建立据点

60-79   ⭐⭐    战略储备级
        特点: 市场潜力中等，需观察
        建议: 建立合作伙伴关系

40-59   ⭐     风险评估级
        特点: 机会与风险并存
        建议: 试点项目，小额投资

20-39   ⏳    观察等待级
        特点: 市场尚未成熟
        建议: 持续跟踪，等待突破

0-19    ❌    不建议进入
        特点: 风险远高于收益
        建议: 继续观察，短期无进入计划
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 4. 全球市场矩阵分析

### 4.1 矩阵维度

**横轴 (X)**: GWSI 评分 (0-100)
- 反映市场的战略价值
- 得分越高，战略地位越重要

**纵轴 (Y)**: 市场吸引力 (0-100)
- 综合考虑: 市场规模、增长率、利润率
- 可选: 按风电装机容量排序

### 4.2 四象限战略

```
       市场吸引力 (Y)
            ↑
       100  │        Ⅱ  │  Ⅰ
            │ 战略储备  │ 优先进入
            │ (中等投资) │ (大力投资)
        75  │         │
            │─────────┼─────────
        50  │         │
            │ 观察等待  │ 风险评估
            │ (持续跟踪) │ (试点项目)
        25  │         │
            │         │
         0  └─────────┼─────────→ GWSI (X)
            0        50        100
```
